概述

使用本系统能实现对GSK设备的状态监控，通过下文你将了解到如何使用本系统。

准备工作

  • 记录本机电脑IP，用于设置到某些型号CNC。
举例：
192.168.11.19

安装说明

  • 运行“GSK设备管理系统.exe”安装。一般情况下可一直点击“下一步”按默认安装。

 

软件注册

  • 右键“桌面/启动GSK车间管理”，以管理员身份运行。
  • 发送“注册码”给业务人员生成验证码。

  • 填入验证码后按“确定”完成注册。

运行

  • 运行 “桌面/启动GSK车间管理”。

 

CNC配置

支持版本

GSK980TDi配置

操作步骤：

  • 切换页面确认版本号；

  • 切换到“参数->状态参数”页面；

    • 修改0250号参数，置位“RMDATA监控”和“MBT监控”；

  • 切换到IP页面记下IP地址。

 

GSK988TA配置

操作步骤：

  • 切换页面确认版本号；

  • 切换到“设置->CNC设置->系统IP”页面；

    • 修改“IP地址”；示例：
    IP地址：192.168.11.111
    子网掩码：255.255.255.0
    默认网关：192.168.11.254
    • 修改“外部接入IP地址”（即电脑IP）；示例：
    xxxxxxxxxx
    （序号任意）     192.168.11.19   读写

  • 切换到“系统->参数”页面；

    • 修改0930号参数，置位“MODBUS”、“NDSVR”和“RMEN”;

添加设备

点击“添加”按钮后出现：

 

  • 设备类型包括：机床CNC、机器人。
  • IP地址：设备的IP地址。
  • Slave ID: 表示modbus的从站ID，非modbus设备此处可填1。
  • 设备名称：车床1，车床2，切削1、机器人1、机器人2等自定义名称,最长8位字符。
  • 设备型号：例如：机床有GSK988TA，机器人有RB08,最长15位字符。
  • 设备编号：机床对应CNC序列号，最长20字符，例如L05CUM00822TA1H。

成品输出

  • 点击打开设备的“成品输出”，则看板的总产量使用这些设备产量之和。

删除

  • 不连接设备点击“删除”。

编辑

  • 编辑设备点击“编辑”。

保存

  • 所有操作必须要点击“保存”才生效。
  • 点击“保存”后需要等待服务器对设备重新连接。

 

设备总览

  • 机床共有4种状态：关机（灰）、待机（黄）、加工（绿）和报警（红）。
  • 关机时间：机床和机器人都能提供，从当天0点开始的关机时间。
  • 开机时间：机器人只能提供开机时间，不能提供开机后具体状态的时间。
  • 待机时间：只机床提供，从当天0点开始，开机且非加工状态的时间。
  • 报警时间：只机床提供，从当天0点开始。
  • 加工时间：只机床提供，从当天0点开始。
  • 加工件数：只机床提供，从当天0点开始。
